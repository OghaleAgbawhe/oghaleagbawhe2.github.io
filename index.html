TENNESEE HOUSING DATASET (KAGGLE) DATA CLEANING - SQL

Performed data scrubbing on a housing dataset found on Kaggle using SQL queries. In this project, I cast pre-existing datatypes into functional datatypes, broke apart mixed data into appropriate columns, handled missing, incomplete and redundant data using CTE's, and updated columns for an intuitive understanding of real estate data.

    • MOVIE INDUSTRY DATA (KAGGLE)
    CLEANSING AND CORRELATIONS - PYTHON

    Conducted a correlational (bivariate) analysis on a dataset with 4 decades of movie data to understand how a movie can achieve high gross earnings. I handled duplicates, missing data, incorrect data types, and repetitive data. I worked with Pandas, Numpy, Seaborn and Matplotlib (Pyplot) to create scatterplots, heatmaps, and correlational matrices (using Pearson, Kendall, and Spearman)

    • TEXT MINING ON AMAZON FOOD REVIEWS
    - PYTHON

    Text mining project focused on food reviews, to extract valuable insights from customer feedback. Upon exploring, cleaning and preprocessing the data, I utilized advanced NLP, deep learning and machine learning techniques to achieve a remarkable 92% accuracy in sentiment classification. Models employed include Logistic regression, Naïve Bayes, Gated Recurrent Unit (GRU) and Long Short-Term Memory (LSTM).
